# Skills locais

Este diretorio guarda memoria operacional do projeto. Cada skill fica em uma
pasta com um `SKILL.md` curto e acionavel.

## Skills disponiveis

- `sage-proof-pipeline/`: regenerar ou ajustar a sequencia CM/Stage 16 com
  WSL, Sage e Conda.
- `small-unit-graph-search/`: trabalhar no exercicio de ate 32 pontos.
- `repo-maintenance/`: Git, GitHub, Git LFS e higiene de artefatos grandes.

## Convencao

- A descricao no topo deve deixar claro quando usar a skill.
- `Gotchas` deve registrar falhas reais deste repositorio, nao dicas genericas.
- Referencie arquivos reais em vez de duplicar conteudo longo.
- Atualize a skill quando um erro recorrente aparecer no fluxo de trabalho.
