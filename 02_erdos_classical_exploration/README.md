# Erdos Classical Exploration

Exploracao da tecnica Erdos-classica e de referencias externas ligadas a
construcoes finitas densas de distancias unitarias.

## Conteudo

- `unit_distance_projection_vs_grid.py`: comparacao didatica entre uma projecao
  booleana em 2D e uma grade regular com o mesmo numero de pontos.
- `EXTERNAL_APPROACHES_REVIEW.md`: avaliacao de repositorios externos e ideias
  visuais correlatas, incluindo uma construcao finita baseada em
  `Z[i, rho]`.
