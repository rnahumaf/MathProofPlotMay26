# unit-distance-problem

Visualizacoes e buscas finitas para o problema de distancias unitarias no
plano. O repositorio esta dividido por linha de investigacao; cada pasta contem
seus scripts, dados e figuras representativas.

Referencia principal para a construcao CM/Stage 16: Will Sawin, *An explicit
lower bound for the unit distance problem*, arXiv:2605.20579v1, 20 May 2026:
https://arxiv.org/html/2605.20579v1

## Mapa

| Pasta | Conteudo |
| --- | --- |
| `01_openai_theorem_exploration/` | Exploracao da construcao CM/ciclotomica inspirada na prova de 2026, incluindo Stage 5-16, galerias finais e editor visual. |
| `02_erdos_classical_exploration/` | Exploracao da linha Erdos-classica e referencias externas comparativas. |
| `03_uccs_exploration/` | Runs UCCS, estatisticas por quadrado e artefatos visuais da busca por fecho de circulos unitarios. |
| `04_max_proven_to_n21/` | Comparacao com maximos comprovados conhecidos ate `n = 21` e buscas pequenas auxiliares. |
| `05_n25_proof_search/` | Investigacao e busca de uma prova para `n = 25`, incluindo certificados, filtros e subcores. |

Arquivos de agente e memoria operacional, como `AGENTS.md`, `GUIDELINES.html`
e `SKILLS/`, sao mantidos localmente para trabalho interno e nao fazem parte do
historico versionado pretendido.
